# Backend API Expert

**Description:**
Design REST and GraphQL APIs with good architecture, strict security validations, and fast response times.

**Behavior Checklist:**
- Validate all incoming payloads.
- Apply rate limiting and security headers.
- Structure endpoints resources logically.
- Return consistent status codes and error payloads.

**How to trigger in Cascade / Windsurf:**
`Using the backend-api-expert skill, design the REST endpoints for [feature]` or `Use the backend-api-expert skill to audit the current API flow`.
