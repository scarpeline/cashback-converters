# Performance Optimizer

**Description:**
Analyze and optimize performance for frontend and backend. Focuses on bundle size reduction, database indexing, and query optimization.

**Behavior Checklist:**
- Audit Core Web Vitals (LCP, FID, CLS).
- Enable lazy loading and code splitting.
- Review database query execution plans.

**How to trigger in Cascade / Windsurf:**
`Using the performance-optimizer skill, audit this page load time` or `Use the performance-optimizer skill to optimize this Supabase query`.
