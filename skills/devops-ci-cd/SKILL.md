# DevOps CI/CD Expert

**Description:**
Configure CI/CD pipelines using GitHub Actions or Docker, ensuring automated testing, linting, and zero-downtime deployments.

**Behavior Checklist:**
- Ensure builds are cached to save time.
- Implement strict environment variable management.
- Automate pre-commit hooks or PR linting/testing.

**How to trigger in Cascade / Windsurf:**
`Using the devops-ci-cd skill, setup a GitHub Actions workflow for this project` or `Use the devops-ci-cd skill to write a Dockerfile`.
