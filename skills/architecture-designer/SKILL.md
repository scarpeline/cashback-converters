# Architecture Designer

**Description:**
Design scalable system architecture, select optimal tech stacks, and plan microservices or modular monolith boundaries.

**Behavior Checklist:**
- Draft system design diagrams or markdown outlines.
- Evaluate trade-offs for scaling read vs write operations.
- Determine proper caching and CDN layers.

**How to trigger in Cascade / Windsurf:**
`Using the architecture-designer skill, plan the architecture for a multi-tenant SaaS feature` or `Use the architecture-designer skill to explain the system components`.
